# 响应式随笔便笺 (Responsive Notes App)

一个使用 HTML5、Tailwind CSS 和原生 JavaScript 开发的响应式随笔便笺网页应用，支持多设备适配和本地数据存储。

## ✨ 功能特性

### 📱 响应式布局
- **PC端**：采用多列网格布局（每行3-4张卡片）
- **移动端**：自动切换为单列布局，卡片宽度自适应屏幕
- **导航栏**：顶部固定标题栏，新建按钮位置根据设备自适应

### 🎨 交互功能
- **新建随笔**：点击按钮弹出模态框/抽屉式弹窗
- **弹窗适配**：PC端固定宽度模态框，移动端抽屉式弹窗
- **数据存储**：使用 localStorage 本地存储，数据持久化
- **排序规则**：新保存的随笔显示在最前面（按时间倒序）

### 💎 视觉设计
- 使用 Tailwind CSS 极简风格
- 卡片设计：细微阴影、圆角、柔和浅色背景
- 按钮反馈：点击时有缩放和颜色变化效果
- 平滑动画：模态框淡入、卡片出现动画

## 🛠️ 技术栈

- **HTML5**：语义化标签，合理的页面结构
- **Tailwind CSS**：响应式设计，快速UI开发
- **原生JavaScript**：清晰的代码结构，逻辑与UI分离
- **localStorage**：本地数据持久化存储

## 🚀 快速开始

### 方法一：直接打开HTML文件

1. 克隆或下载项目代码
2. 找到 `index.html` 文件
3. 直接用浏览器打开该文件

### 方法二：使用本地HTTP服务器（推荐）

#### 使用 Python
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

#### 使用 Node.js
```bash
# 安装 http-server（如果未安装）
npm install -g http-server

# 启动服务器
http-server -p 8000
```

#### 使用 PHP
```bash
php -S localhost:8000
```

4. 在浏览器中访问：`http://localhost:8000`

## 📖 使用说明

### 1. 新建随笔
- **PC端**：点击右上角的「新建随笔」按钮
- **移动端**：点击右下角的悬浮圆形按钮（+）

### 2. 编辑内容
- 在弹出的模态框/抽屉中输入标题和内容
- 标题为可选，内容为必填

### 3. 保存随笔
- 点击「保存」按钮
- 新随笔会自动显示在列表的最顶部

### 4. 删除随笔
- 点击随笔卡片右下角的删除图标
- 在确认对话框中点击「确定」

### 5. 查看随笔
- 所有随笔按时间倒序排列
- 支持在不同设备上查看和编辑

## ⚠️ 注意事项

### 数据存储
- 随笔数据存储在浏览器的 localStorage 中
- 数据仅保存在当前设备的当前浏览器中
- 清除浏览器数据会导致随笔内容丢失
- 换设备后无法访问原设备上的随笔

### 功能限制
- 不支持跨设备数据同步
- 单个浏览器域名的 localStorage 容量约为 5MB
- 不支持富文本编辑
