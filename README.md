# 响应式随笔便笺 (Responsive Notes App)

一个使用 HTML5、Tailwind CSS 和原生 JavaScript 开发的响应式随笔便笺网页应用，支持多设备适配和后端 API 数据存储。

## ✨ 功能特性

### 📱 响应式布局
- **PC端**：采用多列网格布局（每行3-4张卡片）
- **移动端**：自动切换为单列布局，卡片宽度自适应屏幕
- **导航栏**：顶部固定标题栏，新建按钮位置根据设备自适应

### 🎨 交互功能
- **新建随笔**：点击按钮弹出模态框/抽屉式弹窗
- **编辑随笔**：点击卡片上的编辑图标修改内容
- **删除随笔**：点击卡片上的删除图标移除内容
- **弹窗适配**：PC端固定宽度模态框，移动端抽屉式弹窗
- **数据存储**：通过后端 API 与数据库交互，数据持久化存储
- **排序规则**：新保存的随笔显示在最前面（按时间倒序）

### 💎 视觉设计
- 使用 Tailwind CSS 极简风格
- 卡片设计：细微阴影、圆角、柔和浅色背景
- 按钮反馈：点击时有缩放和颜色变化效果
- 平滑动画：模态框淡入、卡片出现动画

## 🛠️ 技术栈

### 前端
- **HTML5**：语义化标签，合理的页面结构
- **Tailwind CSS**：响应式设计，快速UI开发
- **原生JavaScript**：清晰的代码结构，逻辑与UI分离

### 后端
- **Node.js**：JavaScript运行环境
- **Express**：Web应用框架
- **MySQL**：关系型数据库
- **RESTful API**：前后端通信接口

## 🚀 快速开始

### 前提条件
- 安装 Node.js 和 npm
- 安装 MySQL 数据库

### 1. 启动后端服务器

```bash
# 进入后端目录
cd ../essays_admin

# 安装依赖
npm install

# 配置环境变量
# 编辑 .env 文件，设置数据库连接信息

# 启动后端服务器
node app.js
```

### 2. 启动前端服务器

```bash
# 返回前端目录
cd ../essays_ui

# 使用 http-server 启动
npx http-server -p 8000
```

### 3. 访问应用

在浏览器中访问：`http://localhost:8000`

### 4. 环境变量配置（.env）

在 `essays_admin` 目录下创建 `.env` 文件，配置以下内容：

```env
PORT=3000
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_password
DB_NAME=essays_db
```

**注意**：
- 确保 MySQL 服务已启动
- 首次启动后端服务器时，会自动创建数据库和表结构
- 默认会添加两条测试数据

## 📖 使用说明

### 1. 新建随笔
- **PC端**：点击右上角的「新建随笔」按钮
- **移动端**：点击右下角的悬浮圆形按钮（+）

### 2. 编辑随笔
- 点击卡片右下角的编辑图标（蓝色铅笔）
- 在弹出的模态框中修改标题和内容

### 3. 删除随笔
- 点击卡片右下角的删除图标（红色垃圾桶）
- 在确认对话框中点击「确定」

### 4. 保存内容
- 在模态框中输入标题和内容后，点击「保存」按钮
- 标题为可选，内容为必填

### 5. 查看随笔
- 所有随笔按创建时间倒序排列
- 支持在不同设备上查看和编辑
- 数据通过后端 API 实时同步

## ⚠️ 注意事项

### 数据存储
- 随笔数据通过后端 API 存储在 MySQL 数据库中
- 数据持久化存储，支持跨设备访问
- 清除浏览器数据不会影响数据库中的数据

### API 接口
- 后端 API 运行在 http://localhost:3000/api
- 提供完整的 CRUD 接口（新增、查看、编辑、删除）
- 支持跨域请求

### 功能限制
- 目前不支持用户认证系统
- 不支持富文本编辑
- 建议在现代浏览器中使用
